# local-dev-docker-nuxtjs

## How to use

```
$ git@github.com:nitncwind-org/local-dev-docker-nuxtjs.git

$ cd local-dev-docker-nuxtjs

$ git clone git@github.com:nitncwind-org/{nuxt-project}.git

$ cp .env.sample .env

$ vim .env
dir=nuxt-project

$ docker-compose up -d
```


### vue.js ver

```
$ git@github.com:Yamada-Factory/local-dev-docker-nuxtjs.git

$ cd local-dev-docker-nuxtjs

$ git clone git@github.com:{vue-project}.git

$ cp .env.sample .env

$ vim .env
dir=nuxt-project

$ vim docker-compose.yml
- command: sh -c "npm install && npm run dev"
+ command: sh -c "npm install && npm run serve"

- - "3000:3000"
+ - "8080:8080"

$ docker-compose up -d
```
